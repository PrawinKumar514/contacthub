# 📇 ContactHub

A modern Contact Management System built with **Next.js 15**, **MongoDB Atlas**, **TypeScript**, **Tailwind CSS**, and **ShadCN UI**. ContactHub helps users efficiently manage personal and professional contacts with a clean, responsive, and intuitive interface.

## ⚙️ Installation Guide

### Clone Repository

```bash
git clone https://github.com/PrawinKumar514/contacthub.git
cd contacthub
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file in the root directory.

```env
MONGODB_URI=your_mongodb_connection_string
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

## 🌐 MongoDB Atlas Setup

1. Create MongoDB Atlas Cluster
2. Create Database User
3. Configure IP Access List
4. Copy Connection String
5. Add Connection String to `.env.local`

Example:

```env
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/contacthub
```
